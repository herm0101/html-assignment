List of HTML Tags and Their Descriptions

<h1>
Defines the most important heading. There are six levels of headings in HTML, <h1> to <h6>.
<p>
Defines a paragraph of text.
<a>
Defines a hyperlink, which is used to link one page to another.
<img>
Defines an image in an HTML document.
<ul>
Defines an unordered list, which is a list of items in no particular order.
